[审核中] Windows 平台录屏软件 Bandicam 配置指南

本文最后更新于：2 天前

安装 Bandicam

  1. 下载地址：[LRepacks] [423Down]
  2. 运行安装程序按照提示安装即可

设置 Bandicam

  1. 如果电脑安装了 NVIDIA 显卡，那么首次打开软件会提示是否启用 NVIDIA NVENC，点击 是 即可
  2. 设置软件界面为中文。

    在弹出的对话框点击 是 将自动重启软件设置显示语言。
  3. 可选是否设置输出文件夹。
  4. 设置录制参数。

    这里也可以添加图片水印或文字水印，我就不设置了，大家可以自行研究一下，也不难。
  5. 最好设置 录制前显示倒计时 3 秒。
  6. 设置录像格式。

    最好全部按照上面的参数进行设置才能保证最优录像质量。
    至于 FPS，设置一个自己想要的高刷即可 (可以使录制出的画面流畅)。Bandicam 好就好在可以录制帧率高于当前屏幕刷新率的视频。不会占用太多的 GPU，我的 GTX 650 Ti 都没问题。

开始录屏

  1. 然后就可以使用之前设置的快捷键开始录屏了 (注意暂停和停止的快捷键不要搞混了)，Bandicam 支持自定义录制区域，跟随鼠标等录屏样式。
  2. 录屏完成后可以通过软件界面右上角的文件夹图标就可以快速打开保存文件夹了。
