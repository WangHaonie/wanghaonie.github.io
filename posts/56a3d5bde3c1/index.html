安装所需软件

安装 Visual Studio Code

注意：如果你电脑中已经安装了 Visual Studio Code 或者是其他源代码编辑器的话就可以跳过这一步了

  1. 进入 Visual Studio Code 官网，然后选择下载 System Installer x64 可以让 Visual Studio Code 默认以管理员身份运行。
  2. 下载完成后，打开安装程序勾选我同意此协议，然后点击下一步。
  3. 安装位置保持默认，点击下一步。
  4. 创建开始菜单文件夹也保持默认，点击下一步。
  5. 勾选如图所示的两个框框，然后点击下一步。
  6. 点击安装，等待安装完成。
  7. 安装完成后勾选运行 Visual Studio Code，然后点击完成就会打开 Visual Studio Code，我们需要进行一些简单的配置。
  8. 点击左侧的扩展程序图标，然后搜索 chinese 然后点击第一个 (一般都是第一个)，再点击 Install 安装汉化包。
  9. 稍等片刻后在右下角会出现一个弹窗，点击 Change Language and Restart之后，Visual Studio Code 会重启并切换为简体中文。

安装 Node.js

  1. 进入 Node.js 官网，一般下载长期维护版，因为我们后面只是用这个软件来下载和安装一些有关博客网站的包，对功能没什么特殊需求，所以没必要用最新尝鲜版。
  2. 打开下载好的安装包之后，点击 Next。
  3. 勾选 I accept the terms in the License Agreement，然后再点击 Next。
  4. 安装路径保持默认，点击 Next。
  5. 确保这些组件都勾选安装之后，然后点击 Next。
  6. 这个界面直接点 Next 跳过，不用勾选那个框框。
  7. 现在就可以点击 Install 来进行安装了。
  8. 显示这个界面就说明 Node.js 安装成功，点击 Finish 退出安装程序。

安装 Git for Windows

  1. 进入 Git for Windows 官网，这里有很多下载选项，分别是 Git for Windows 的32位和64位安装版，以及32位和64位的便携版。由于现在大多数电脑系统都是64位并且为了方便后期不用配置环境变量，直接使用git来运行命令的话，所以这里要选择 64-bit Git for Windows Setup。
  2. 下载完成后打开 Git for Windows 安装包，点击 Next。
  3. 同样安装位置还是保持默认，点击 Next。
  4. 如果你正在使用 Windows 10 及以上的系统，并且已经安装了 Windows 终端 (点此安装) 的话，我们可以勾选这个 And a Git Bash Profile to Windows Terminal 以便我们可以在 Windows 终端中快速打开 Git Bash，其他选项默认即可，然后点击 Next。
  5. 创建开始菜单文件夹也保持默认，点击下一步。
  6. 关于默认编辑器这里的话就选 Visual Studio Code，然后点击 Next。
  7. 这之后的几个界面都直接点 Next。








  8. 然后点击 Install 进行安装。
  9. 取消勾选这两个框框，然后点击 Finish 即可退出安装程序。

配置远程上传

创建 GitHub Pages

  1. 首先登录到 GitHub，如果你没有 GitHub 账号的话，那你就要先注册一个账号，要用邮箱注册，具体过程这里就省略了，按照网站提示注册就行。
  2. 注册完成后后来到首页，在右上角有一个加号旁边的小三角形，然后点一下那个三角形选择 New Repository。
  3. 来到这个页面后，第一步在 Repository name 那里必须要输入”你的用户名.github.io“，如果你不知道你的用户名是什么，在这个输入框的前面有一个 Owner，它下面的名字就是你的用户名。第二步 Description 那里是对这个仓库的描述，可以随便写，也可以不写。注意下方的的 Public 必须要选上，以确保这个仓库是对外公开的，不然的话别人就无法访问这个你的网站了。然后在下面有一个 Add a README file，这个也是对仓库的描述，只不过你之后就可以在这个文件里面写更多有关该仓库的内容，可以不用勾。最后就点击最下面绿色的那个 Create repository。
