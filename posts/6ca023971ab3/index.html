(验证中...) Windows 系统软件推荐

本文最后更新于：2 天前

系统运行库

.NET Framework

  1. .NET Framework 4.8
    下载地址：[Microsoft 官网] [简体中文语言包 - Microsoft 官网]
  2. .NET Framework 3.5
    下载地址：[Microsoft 官网]

DirectX Runtime

  1. DirectX 修复工具
    下载地址：[423Down]

Microsoft Visual C++ Runtimes (下列任选一个下载)

  1. 多合一安装包
    下载地址：[Github] [WorkUpload] [LRepacks]
  2. 微软常用运行库合集
    下载地址：[423Down]

驱动管理

360 驱动大师单文件版

  1. 下载地址：[423Down]
  2. 已阉割恶心功能，放心使用。

IObit Driver Booster

  1. 下载地址：[LRepacks] [423Down]
  2. 最专业的的驱动管理软件，能科学上网的话优先选他
  3. 貌似可以通过改 hosts 访问：68.232.45.253 www.cd4o.com，不行的话去 站长之家 找找。

    为什么不推荐网卡版：
    1. 现在装系统普遍都是 Windows 10 以上，自带网卡驱动。
    2. 装的不是 Windows 10 也不要紧，现在 安卓、苹果 手机都支持数据线共享网络给电脑。

网络浏览器

Microsoft Edge

  1. 下载地址：[Microsoft 官网]
  2. 以前都觉得 Google Chrome 浏览器好用，真的是不用不知道，一用吓一跳，Microsoft Edge 基于和 Google Chrome 同样版本的 Chromium 内核，其功能竟然比 Google Chrome 多得多。
  3. 本人觉得很好的功能：
    (1) 启动增强：在开机时后台预加载扩展程序，加快浏览器启动速度，防止像 Google Chrome 那样扩展程序一旦增多，启动速度也慢了
    (2) 退出浏览器时自动清理浏览记录；
    (3) 具有完美的可替代性：服务器在中国有部署，不用科学上网也能登录同步用户数据；支持从 Chrome 应用商店 安装扩展程序；支持从 Google Chrome 已有的用户数据中导入书签，密码，扩展程序等；内嵌式调用 Internet Explorer 浏览器，可在线玩 4399 (前提是要先安装 Flash，再在 IE 里面启用 Flash 加载项才可以)，完美替代某数字那几个浏览器。

Google Chrome

  1. 下载地址：[Google 中国官网] [Google 官网]
  2. 不安装扩展程序 (或安装得少) 的、用不惯 Microsoft Edge 的，可以用 Google Chrome，界面干净，纯净，启动速度快。

    不要用某数字浏览器了，不仅内核不是最新的，而且启动时有些时候会卡白，不知道怎么回事，主要还是广告多，恶心。

解压缩软件

7-zip

  1. 下载地址：[官方网站]
  2. 界面干净，体积小，支持格式多

Bandizip

  1. 下载地址：[官方网站] [Bandizip 7.xx 破解程序 - 蓝奏云]
  2. 界面好看，支持右键菜单压缩文件预览 (最好把这个关了，遇到大文件时会卡，还会使 Windows 10 的黑暗主题失效)，支持智能解压 (判断解压后的文件应该直接出现在当前目录还是新建一个文件夹里)

图片查看器

Microsoft 照片 (仅限 w10/11)

  1. 下载地址：[Microsoft Store]
  2. 没啥好说的，常见格式都能打开

IrfanView

  1. 下载地址：[IrfanView 官网] [IrfanView - 中文语言包]
  2. 支持格式多，界面整洁，启动速度快。
  3. 常用的快捷键：Ctrl+鼠标滚轮 放大缩小图片，L键 翻转图片。

媒体播放器

PotPlayer

  1. 下载地址：[PotPlayer 官网] [PotPlayer 全球官网] [PotPlayer 中国官网]
  2. 没啥好说的，他可是在 B 站上出了名的音视频播放器。

K-Lite Codec Pack

  1. 下载地址：[官方网站] [LRepacks]
  2. 可搭配 PotPlayer 做音视频解码器，当 PotPlayer 遇到不支持的编码格式时，安装它来解决。
  3. 美中不足的就是安装程序设置步骤太复杂，还是用 LAV Filters 吧。

    温馨提示：K-Lite Codec Pack 自带了 LAV Filters，好像不是最新版，但也能用。

安全软件

火绒安全软件

  1. 下载地址：[官方网站]
  2. 无理由推荐，没有之一。

办公软件

Microsoft 365

  1. 下载地址：[官方网站] [本站 - 没有 Microsoft Office 订阅？教你使用黑科技白嫖 Office 365 并激活]
  2. 要求外观过渡动画流畅的同学就选它，需要云备份/存储文档的还是用 WPS 好了。

PDF 文档

Adobe Acrobat DC

  1. 下载地址：[官方网站] [受版权方要求，故删除第三方链接]
  2. PDF 文档格式都是 Adobe 家的发明的，所以无理由使用 Acrobat。

SumatraPDF

  1. 下载地址：[官方网站]
  2. 免费，小巧，启动快，秒开大文件。

文件下载器

Internet Download Manager

  1. 下载地址：[官方网站] [官方网站 - 中国分站] [LRepacks]
  2. 高速内核、多线程下载跑满带宽、自定义 UA，可惜不支持磁力链接

Chromium 内核浏览器原生下载引擎

  1. 指 Microsoft Edge、Google Chrome 等保留了原生下载引擎的浏览器，不是用迅雷下载引擎的某数字浏览器。
  2. 开启一个叫 Parallel downloading 的实验性功能即可开启多线程下载 (有些时候还是没有 IDM 强)。
  3. Edge 浏览器在地址栏输入 edge://flags/，Chrome 浏览器地址栏输入 chrome://flags/，进去后在搜索框内输入 parallel downloading 就可以看到了，把右边的按钮设置为 Enabled，在弹出的提示中点击 Relaunch 按钮重启浏览器就好了。

BitComet (比特彗星)

  1. 下载地址：[423Down]
  2. 支持磁力链接、种子下载，内置几百个 Tracker 服务器加速下载。
  3. B 站上有位 UP 把它用来下载 HTTPS 链接，也就是说可以媲美 IDM，具体怎样看视频

录屏软件

Bandicam

  1. 下载地址：[LRepacks] [423Down]
  2. 配置教程：本站 - PC 端录屏软件 Bandicam 配置指南

    个人总感觉 OBS 录屏上手难，所以未推荐。

软件卸载

IObit Uninstaller

  1. 下载地址：[LRepacks] [423Down]
  2. 这三个软件都有相同的功能：批量卸载软件、清除残留。但这个软件在 Windows 11 上可卸载安卓子系统的应用程序，其他功能如后期软件残留扫描等也是下面两个没有的。

Uninstall Tool

  1. 下载地址：[LRepacks] [423Down]
  2. Uninstall Tool 就是基于 Geek Uninstaller 的代码，所以两者均有相同的功能，就界面来看的话，Uninstall Tool 的界面显得更大方，具体用哪个就看个人爱好了。

Geek Uninstaller

  1. 下载地址：[423Down]

磁盘分区

傲梅分区助手

  1. 分区助手
    下载地址：[官方网站]
    使用码：1122 (一直都是这个，不用去关注公众号)
  2. 分区助手技术员版
    下载地址：[LRepacks] [423Down]

DiskGenius

  1. 中国版
    下载地址：[官方网站]
  2. 国际版
    下载地址：[LRepacks]

搞机必备 (只推荐常用的，实际上肯定不止以下这些)

Everything

  1. 下载地址：[官方网站] [423Down]
  2. 快速搜索计算机上的文件，快到你输完文件名就已经搜索完了。可以集成到资源管理器上下文菜单，右键点击想在里面搜索文件的文件夹，选择 搜索 Everything… 进行搜索，再在设置里面加个快捷键 Ctrl+Alt+E 随时随刻打开搜索。

FileZilla

  1. 下载地址：[官方网站]
  2. FTP 协议传输文件，配合安卓端的 MT 管理器里面的远程管理功能就可以实现局域网内手机电脑文件互传。

TrafficMonitor

  1. 下载地址：[Github]
  2. 支持悬浮窗显示当前上下行网速，任务栏窗口显示 CPU、显卡占用、温度等多个数值。

WinDbg

  1. 下载地址：[Microsoft Store]
  2. 通过分析系统错误内存转储文件来判断具体蓝屏原因，详细教程：知乎

N_m3u8DL

  1. 下载地址：[Github]
  2. 下载 M3U8 视频链接，双击打开粘贴链接使用，也可以配合作者提供的带界面的 SimpleG 来使用，就是在 Github Releases 下面就有一个叫 N_m3u8DL-CLI_with_ffmpeg_and_SimpleG.zip 的文件。

USB Safely Remove

  1. 下载地址：[LRepacks]
  2. 安全弹出 U 盘的，点击托盘图标再点击你的 U 盘图标就可以在不结束进程的情况下解除占用并弹出 U 盘，如果不小心点错导致被弹出的话，再次点击 U 盘图标时，还能弹回来，这时你就又可以使用 U 盘了，比系统自带的好用

小丸工具箱

  1. 下载地址：[官方网站 (可能已失效)] [鹏少资源网]
  2. 高效率压缩视频，CRF 值越大视频压缩质量越低，推荐设置 20~25。

SpleeterGui

  1. 下载地址：[官方网站]
  2. 音频人声分离工具

Ventoy

  1. 下载地址：[Github]
  2. U 盘多启动支持，不用一次又一次的刷 Windows 安装镜像了，也支持各种 ISO 格式，可以启动 PE，非常好用，具体教程 B 站一搜一大堆

Best Trace

  1. 下载地址：[官方网站]
  2. IP 地址查询，CDN 网络节点查询，路由跟踪，Ping 延迟测试等多种网络功能，由 ipip.net 提供 API 支持

IObit Smart Defrag

  1. 下载地址：[LRepacks] [423Down]
  2. 专业的磁盘碎片整理程序，支持固态/机械硬盘。

Total Commander

  1. 下载地址：[423Down (注意还要在下面下载破解文件哦)]
  2. 像个文件管理器，有很多的高级功能，我比较喜欢里面的批量重命名，左键或按住 Ctrl 或 Shift 选择文件，右键取消选择，使用 Ctrl+M 打开批量重命名菜单按照提示输入正确的表达式即可，若弄错了还可以撤销。

磁力资源搜索助手

  1. 下载地址：[官方网站]
  2. 内置多个磁力搜索网站同时搜索磁力链接

Registry Finder

  1. 下载地址：[官方网站]
  2. 顾名思义，查找注册表项、值，比系统自带的好用。

Spacedesk

  1. 下载地址：[官方网站]
  2. 局域网内通过 WiFi 让手机为电脑拓展显示器，可手动设置分辨率，刷新率等
  3. 推荐千兆以上的路由器、局域网使用，否则会卡顿

Resource Hacker (ResHacker)

  1. 下载地址：[官方网站] [423Down]
  2. 免费资源查看器，资源编译器和反编译器的资源编译工具，可以修改可执行文件的版本，作者，图标等参数

Process Monitor (Procmon)

  1. 下载地址：[官方网站] [423Down]
  2. 微软自家的进程监视器，查看每个进程进行的操作。

RAMMap

  1. 下载地址：[官方网站]
  2. 微软自家的内存分析工具，可以在软件界面点击清空系统工作集来清理运行内存。

UPX

  1. 下载地址：[Github] [423Down]
  2. 使用 UPX 算法压缩 exe 文件减小体积，不一定对使用 exe 文件都适用

NatTypeTester

  1. 下载地址：[Github]
  2. 检测当前网络的 NAT 类型，最好先关闭各种代理软件。

UltraISO

  1. 下载地址：[423Down]
  2. ISO 修改器，可以将镜像刷入 U 盘等多种功能。

Axialis IconWorkshop

  1. 下载地址：[LRepacks]
  2. 图标制作器或从 exe，dll 里面提取 ico 图标，教程自行百度。

Nsudo

  1. 下载地址：[Github]
  2. 使用系统权限运行程序，一键告别权限不足对话框
  3. x64文件夹里面的 NsudoLG.exe 是带界面的 Nsudo，NsudoLC.exe 是命令行的 Nsudo，可参考 官方文档 实现直接调用。
  4. 可以使用这个辅助工具将 Nsudo 集成到右键菜单方便直接调用
