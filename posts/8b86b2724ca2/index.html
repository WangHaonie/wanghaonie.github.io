(验证中...) Windows 系统安装教程

本文最后更新于：1 天前

安装方式以及优化项目均为个人发掘，仅供参考。

Windows 镜像自备

请按照以下步骤进行安装，具体设置自行百度

设置 BIOS

若主板没有个别设置项则跳过

  1. 开启 CPU 睿频；
  2. 开启 CPU 虚拟化技术 VT-x、VT-d、AMD-V 等；
  3. 为高频内存启用 XMP；
  4. 更改 SATA 工作模式为 AHCI；
  5. 开启 CSM，并设定 Legacy / UEFI 双启动模式；
  6. 开启 TPM；
  7. 关闭主板自带的电源管理 (防止关机后 USB 口还在供电导致的键盘鼠标常亮)。

安装系统

1.推荐使用 Edgeless、WePE 等纯净的第三方 PE 系统进项安装部署，启动盘的话用 Ventoy 制作，尽量不要使用其他向老毛桃、大白菜等 “捆绑型” PE (主要是考虑到有捆绑软件)。微软官方的 PE 的话偶尔会在分区或在其他的时候报各种错误，不方便修复
2.确保选择的 PE 系统中装有 DiskGenius、WinNT Setup 软件

小提示：对于待安装的是 Windows 7 以上的系统，只要主板支持 UEFI (Intel CPU 3 代以上均支持) 就用 UEFI 引导，没有为什么。

UEFI 引导篇

操作 DiskGenius：

  1. 打开 DiskGenius 软件；
  2. 在左侧右键单击待安装的系统的硬盘的型号，若有分区选择 删除所有分区；
  3. 在左侧右键单击待安装的系统的硬盘的型号，选择 转换为 GPT (GUID) 分区表；

    有人可能会说 UEFI 可以引导 MBR，只有大于 2TB 的硬盘才用 GPT。其实这种说法本人也不是搞的特别清楚，按照 BIOS+MBR、UEFI+GPT 来就行了。而且在 DiskGenius 中，你会发现 MBR 分区表的硬盘无法建立 ESP 分区，也就是说，你要用 UEFI 引导，就必须要 GPT 分区。

  4. 在上方右键单击该硬盘的分区，选择创建 ESP 分区，在弹出的对话框直接点确定；
  5. 在上方右键单击该硬盘的未分配分区，选择 创建新分区，卷标设置为 OS，点击确定。(不要问为什么不设置 4K 对齐，因为 DiskGenius 一般默认就是 4K 对齐，扇区 2048、4096 都是 4K 对齐)
  6. 点击软件界面左上方的 保存更改，等待操作自动完成。

操作 WinNT Setup：

  1. 打开 WinNT Setup 软件
  2. 安装映像位置 选择预先准备的 ISO 或已挂载的 ISO 的 sources\install.wim|esd 文件
  3. 可引导驱动器 点击 展开 下拉列表，根据待安装的硬盘选择带有 EFI System (ESP) 的分区

    在这里有人可能点击的是 选择 按钮，直接跳转到 资源管理器 去选择了，也可以，但是有些时候 资源管理器 并不会显示 ESP 分区，所以才点击 WinNT Setup 自带的下拉列表，如果没有的话，说明软件版本太低，请到 官网 提供的 MediaFire 下载地址 下载最新版本。

  4. 安装驱动器的位置同样 展开 下拉列表，选择对应分区即可。
  5. 选项里面的 版本 推荐选择 专业版 (Windows 10/11) (Windows 7 选旗舰版)，或按需选择。
  6. 优化调整按照下图设置，然后关闭该窗口

  7. 点击 安装，再在弹出的对话框直接点确定就可以开始安装。

Legacy (BIOS) 引导篇

操作 DiskGenius：

  1. 打开 DiskGenius 软件；
  2. 在左侧右键单击待安装的系统的硬盘的型号，选择 重建主引导记录 (MBR)，在弹出的对话框点击是；
  3. 在上方右键单击该硬盘的分区，选择创建新分区，卷标设置为 OS，点击确定；
  4. 再次右键单击该分区，选择 激活当前分区；

    正常情况下重建完主引导记录，创建新分区时，分区就已经自动激活，已激活的分区的文件系统 NTFS 字样显示为红色，同时分区后面也会标注 活动的(Active) 字样，如下图

  5. 点击软件界面左上方的 保存更改，等待操作自动完成。

操作 WinNT Setup：

  1. 打开 WinNT Setup 软件
  2. 安装映像位置 选择预先准备的 ISO 或已挂载 ISO 的 sources\install.wim|esd 文件
  3. 可引导驱动器 点击 展开 下拉列表，根据待安装的硬盘选择先前在 DiskGenius 创建的分区

  4. 安装驱动器的位置同样 展开 下拉列表，选择与 可引导驱动器 相同分区即可。

  5. 选项里面的 版本 推荐选择 专业版 (Windows 10/11) (Windows 7 选旗舰版)，或按需选择。
  6. 优化调整按照下图设置，然后关闭该窗口。最好只按照我的来勾选，其他选项可能不适合 Windows 最新版本

  7. 点击 安装，再在弹出的对话框直接点确定就可以开始安装。

OOBE 设置

网上有很多跳过 OOBE 的方法，个人觉得这个流程没必要跳过，就像新手机买回来一样会有首次开机的设置。而且还可借 OOBE 来完成一部分优化。

Windows 7 的 OOBE 太简单了，随便设置一下即可，该章节适合 Windows 8 以上的 OOBE。

  1. 当询问 是否添加第二种键盘布局 时 添加一个 英语(美国) 的 美式键盘。

    目的是打游戏的时候可以使用快捷键 Win+R 切换到 美式键盘，防止在打游戏时跳出中文输入法。

  2. 账户选择 脱机账户，名字随意，反正后期都要切换到内置管理员账户 Administrator

    Windows 11 正式版的 OOBE 只要拔掉网线就可以跳过登录，Dev 版的 OOBE 可以试试 这个。觉得麻烦的直接登个号也可以 (我就这样)，反正最后也会用内置管理员账户。

切换到 Administrator 的好处：
1.对于喜欢搞机的同学来说，某些 bat 脚本或 cmd 之类的程序需要管理员身份运行，切换到内置管理员后所有的程序都是以管理员身份运行，非常的方便。
2.可以直接修改 C:\Program Files 和 C:\Program Files (x86) 的内容，不会弹出权限不足对话框
3.桌面图标不会显示小盾牌，免得又去改注册表消除小盾牌
4.打开某些应用程序不会跳出 用户账号控制 (UAC) 窗口

坏处：
1.在 Windows 8/8.1 和早期 Windows 10 中，内置管理员账户不能打开自带的 UWP 应用程序。其他操作不受影响
2.某些极其个别少数的软件会禁止自己被内置管理员运行。(放心，日常使用的 99% 软件不会这样的，可忽略这条)

  1. 在 OOBE 的最后一页，关闭 量身定制的体验、广告 ID、墨迹书写和键入，点击接受。

    前两个关了不会出现广告，后面那个应该没人会用到这个。本人看过 B 站上面的一些视频，有的 UP 主把这一页所有选项都关闭了，有的 UP 主看都不看一眼直接下一步。其实仔细看看就会发现这里的选项还是挺有用的。


    不难看出，位置 和 查找我的设备 对于喜欢看天气或者地图的同学来说是非常有用的，有的人可能担心会暴露自己的家庭住址，这肯定不会的，因为电脑 (特别是台式电脑，笔记本电脑可能会装配卫星定位功能) 一般不会有卫星定位的功能，所以不必要担心这个问题；然后就是 诊断数据 了，仔细看看好像还真没用，其实，这个涉及到 Windows 预览体验计划 (点击加入)，没有它，就无法获取 Insider Preview 更新 (除非你用不到这个，就可以把它关闭)。

  2. 这个可能是新版 Windows 10 和 Windows 11 才有的界面，直接跳过
  3. Cortana 页面直接选择 以后再说。觉得有用的可以接受它。
  4. 开始漫长的等待。。。。

  2. 接着在开始菜单点击用户头像，选择注销，然后登录 Administrator
  3. 这时又会弹出个 OOBE，按照上面的关闭 量身定制的体验、广告 ID、墨迹书写和键入即可
  4. 再次进入桌面，点击开始，点用户头像，选择更改账户设置，在左边点击 其他用户，在右边点击之前创建的用户选择删除，点击确认后，等待删除完成。
  5. 如果你安装的是 Windows 11，建议现在安装一个叫 StartAllBack 的软件，否则将无法完成第6步，软件安装完后会自动打开界面，主题根据个人爱好选择，这里为了保持 Windows 11 的原生风格，所以选择 正宗11；在左侧点击 开始菜单，关闭 启用增强型开始菜单，不然的话显示的是 Windows 7 的开始菜单样式，也是根据个人爱好决定；左侧点击 任务栏，根据个人爱好勾选 任务栏图标居中，动态透明效果，顺便点击下面的 通知区域图标，在弹出的对话框中勾选 始终在任务栏上显示所有图标和通知，点击左侧的 资源管理器，在这里可以设置资源管理器顶部的样式；在左侧点击 高级，取消勾选两个 存储并显示最近打开的项目和程序，顺便点击后面的两个 清空列表 按钮。
  6. 取消固定任务栏已有的应用图标，关闭任务视图，Cortana，人脉图标；资讯和兴趣可以显示当天的天气，根据个人需求是否打开或关闭，若要打开，也要关闭悬停时打开。将搜索框设置为仅显示图标，关闭悬停时打开。
  7. 取消在开始菜单中已经固定的应用磁贴。
  8. 打开文件资源管理器，双击 查看 选项卡以打开工具栏，点击选项，在弹出的对话框中，将 打开文件资源管理器时打开 的 快速访问 更改为 此电脑 ，在下方 隐私 部分取消勾选两个在快速访问中显示最近使用的文件和常用文件夹 (在 Windows 11 中，还有一个 显示来自 Office.com 的文件 也要关闭)，顺便点击旁边的 清除 按钮，最后点击应用，确定。找到左侧的快速访问列表，将所有固定的项目取消。
  9. 下载并拷贝解压后的 HEU KMS Activator (仅适用于 Windows 10/11 数字激活) 到电脑，打开软件，如图，为新版界面



    点击左侧的数字激活，再点击上方 数字许可证激活 上方的 图标 即可进行自动激活。


  10. 打开文件资源管理器，双击查看选项卡打开工具栏，点击选项，在弹出的对话框中，将 打开文件资源管理器时打开 的 快速访问 更改为 此电脑 ，在下方 隐私 部分取消勾选两个在快速访问中显示最近使用的文件和常用文件夹 (Windows 11 还有一个显示来自 Office.com 的文件 的选项也要关闭)，顺便点击旁边的 清除 按钮，最后点击应用，确定。
  11. 右键点击任务栏，选择 任务栏设置，下滑点击 选择哪些图标显示在任务栏上，打开 通知区域始终显示所有图标，返回，再点击 打开或关闭系统图标 将定位，麦克风关闭。再点击左侧的 开始，将 显示最近添加的应用，偶尔在开始菜单中显示建议，在开始菜单和任务栏的跳转列表中以及文件资源管理器的快速使用中显示最近打开的项 关闭。点击下方的 选择哪些文件夹显示在开始菜单上，将 文档，图片 关闭，仅保留 设置。

    Windows 11 的开始菜单文件夹设置在 设置>个性化>开始>文件夹 里面 (或者打开开始菜单，右键点击选择开始菜单设置>文件夹)，正常情况下这里的选项全部是关闭的，仅打开 设置 即可。

  12. 回到设置主页，点击 系统，在左侧点击 剪切板，在右侧开启 剪切板历史记录。在左侧点击 关于，在右侧选择高级系统设置，在弹出的对话框中选择 硬件 选项卡，点击下方的 设备安装设置，选择 否，点击 保存更改。
  13. 回到设置主页，点击 设备，在左侧点击 自动播放，将 在所有媒体和设备上使用自动播放 关闭
  14. 回到设置主页，点击 更新和安全，左侧点击 传递优化，将 允许从其他电脑下载 关闭

    传递优化会占用硬盘空间，建议关闭

  15. 回到桌面，在桌面上右键点击选择个性化。在左侧点击主题，在右侧点击桌面图标设置 (若当前屏幕分辨率较低，在右侧的选项可能在下面)。在弹出的对话框中将控制面板勾选上，点击应用即可。
  16. 回到桌面，打开 控制面板。选择 硬件和声音，点击 电源选项，在左侧点击 选择关闭显示器的时间。将该页面两个选项时间都设置为 从不。
  17. Win+R 打开运行，输入 regedit 打开注册表编辑器，在上方输入框中直接粘贴 (不清空前面的 计算机 字样)：

\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\DelegateFolders\{F5FB2C77-0E2F-4A16-A381-3E560C68BC83}，按下回车可定位到该项所在位置，在左侧将项 {F5FB2C77-0E2F-4A16-A381-3E560C68BC83} 命名为 -{F5FB2C77-0E2F-4A16-A381-3E560C68BC83}

防止插入可移动设备时，在文件资源管理器会显示重复的图标

  1. Win+R 打开运行，输入 service.msc，在弹出的对话框中下拉找到以 Diagnostic 开头的四个服务和 Superfetch (有可能也叫作 SysMain) 总共 5 个服务全部停止并禁用

Diagnostic 开头的 4 个服务必须关，有时会无缘无故的卡 CPU，Superfetch (SysMain) 会加快软件第二次以后的打开速度，但是并不明显，如果是机械硬盘的话，建议开启，固态硬盘就直接关掉。

  1. 禁止 Windows 更新在更新系统的同时更新驱动程序 (就这个对于老 N 卡用户来说太流氓了，不禁用的话他会把显卡驱动程序更新到最新版本，导致负优化降低游戏帧数，懂的都懂)。点此下载附件

    这里没有提到，但是网上可能提到过的优化项目:

    1.虚拟内存。B 站上面有 UP 主已经解释过了，点击跳转观看，虚拟内存必须开启，不能关闭。因为有些游戏或者软件都要依靠虚拟内存，所以在 Windows 系统中虚拟内存就保持系统默认的自动分配即可，如果要手动设置，必须设置在 C 盘并且必须在系统推荐值的以上。

    2.Windows 更新。在网上总是有些人对 Windows 更新不满意，说什么在工作的时候更新，重启又费时间，有些时候更新完了还会出错，所以对Windows更新的印象简直是糟糕。但其实 Windows 更新没有那么糟糕，只要你能保证你装系统的那天，你的系统镜像是微软发布的最新版本 (不是 Ghost 镜像，可以是 UUP 整合的镜像)，那么你安装进去后，系统更新补丁是很少的，而且现在 Windows 的更新也几乎是有规律了，基本上在一个星期左右更新一次，而且每次基本上就是累积更新 (系统升级更新)、.NET Framework 的累积更新、恶意软件删除工具的更新，这些更新虽然要求重启，但也能在 10 分钟内完成，甚至更短，并没有太大的更新。如果禁用 Windows 更新的话，你的 Windows 版本将随着时间的推移逐渐被官方淘汰，无法体验最新功能 (虽然也不一定每次更新都保证有新功能的加入，但是可以肯定的是微软在更新中修复了许多 Bug，是值得我们用户去更新的，像我这样，每天无聊的时候，我都会手动去检查更新确保 Windows 是最新版本)，最终，你的 Windows，将变成传言中的 Bugdows。如果你当前的系统已经禁用掉了更新，请立马把它打开，检查一下更新，如果你的系统版本号太老 (可自行百度 Windows 的最新版本)，那么不用检查了直接重装到最新的系统，否则又会下载一大堆安装文件。

    Windows 更新的确出过错，但可谓是少之又少，最近的是 2022 年 5 月份的适用于 Windows 11 的 KB5013943 补丁 (升级后系统版本 22000.657)，导致依赖 .NET Framework 的应用程序直接报错 0xc0000135，但是他的解决方法也并不难，你只需要在控制面板里面的 启用或关闭 Windows 功能 里的点 .NET Framework 关闭并重新打开就可以了，整个过程也用不到十分钟。详见 此处。

Windows 7/8/8.1 专区

Windows 7 如果你是用 UEFI+GPT 的引导的话，普通激活工具是无法激活的，推荐 win7uefi_Chew-WGA 百度一搜一大堆.如果用 Legacy 引导的话，可以用 小马激活工具

他可不是什么垃圾软件，他能永久激活 Windows 7，无病毒，无广告。

Windows 8/8.1 不受分区表影响，但是目前找不到对应的可以永久激活的软件，只能 KMS 激活 3 个月再续期 (所以不要用 Windows 8/8.1 作为主力系统啦)。
还有一件重要的事，Windows 8/8.1 使用内置管理员账户会无法打开自带的 UWP 应用。

  1. OOBE 就不用多说了，Windows 7 的随便设置一下就行，Windows 8/8.1 可以参照 Windows 10/11 的来。
  2. 首先，我们还是用在 OOBE 创建的账户进入桌面，点击开始菜单，搜索cmd，在搜索的结果中右键点击cmd，选择以管理员身份运行输入 net user Administrator /active:yes 启用内置管理员账户，然后在开始菜单关机后面选项选择注销，然后再以管理员身份登入。
  3. 还是老规矩，取消固定任务栏上的所有图标。
  4. 右键桌面点击 个性化，在弹出的窗口左侧点击 更改桌面图标，把 控制面板 勾上点击 应用 确定，关闭窗口。
  5. 打开 计算机 (文件资源管理器) 在左侧 收藏夹 那里把所有的文件夹都删除。
  6. 顺手在文件资源管理器的工具栏打开 控制面板，选择 硬件和声音，点击 自动播放，取消勾选 为所有媒体和设备使用自动播放，点击右下角的保存；返回点击 电源选项，选择 关闭显示器的时间 将旁边两个都设为 从不，点击保存更改。
  7. 回到桌面，Win+R 打开运行框，输入 services.msc 将 Homegroup 开头的 2 个服务和 Diagnostic 开头的 3 个服务，往下找到 Superfetch 和 Windows Update 共计 7 个服务全部停止并禁用。
