(重构中...) 小米手机刷入 PixelExperience 类原生系统教程

手机参数

  1. 名称：Xiaomi Mi 11 Lite 5G
  2. 型号：M2101K9C
  3. 代号：Renoir
  4. 处理器：Qualcomm Snapdragon 780G @ 1.90 GHz
  5. 分辨率：2400x1080@90Hz
  6. 运行内存：8 GB
  7. 存储内存：256 GB
  8. 操作系统：MIUI 13 (V13.0.9.0.SKICNXM)
  9. 操作系统内核：Android 12

准备工具

  1. 带有链接的点击即可下载。
  2. 一台搭载了 Windows 7 以上系统的电脑。
  3. 一条全功能的 Type-A 转 Type-C 数据线 (原装数据线)。
  4. 小米手机解锁工具
    https://www.miui.com/unlock/download.html
  5. Google USB 驱动
    https://dl.google.com/android/repository/usb_driver_r13-windows.zip
  6. ADB (版本必须大于或等于 31.0.2，Wiki 说的，我也不知道为什么，最好用最新版吧)
    https://dl.google.com/android/repository/platform-tools-latest-windows.zip
  7. PixelExperience_Plus_renoir-12.1-20220715-1944-OFFICIAL.zip (PixelExperience ROM，建议重命名为 rom.zip 以便输入)
    https://download.pixelexperience.org/renoir
  8. PixelExperience_Plus_renoir-12.1-20220715-1944-OFFICIAL.img (PixelExperience Recovery，建议重命名为 rec.img 以便输入)
    https://download.pixelexperience.org/renoir
  9. vendor_boot.img
    https://gitlab.pixelexperience.org/android/vendor-blobs/wiki_blobs_renoir/-/raw/twelve/twelve/vendor_boot.img
  10. Magisk
    https://magiskcn.com/magisk-download/
  11. payload-dumper-go
    https://github.com/ssut/payload-dumper-go/releases
  12. boot.img (用 payload-dumper-go 解包 ROM 里面的 payload.bin 来获取，然后用 Magisk 去修补)
  13. LSPosed for Zygisk
    https://github.com/LSPosed/LSPosed/releases

刷机前须知

  1. 小米手机的话要在 设置>开发者选项>设备解锁状态 里绑定账号和设备，等待 7 天才能解锁。若绑定失败可尝试退出小米账号重新登录即可。所以对于喜欢搞机的同学来说，买到小米手机第一件事就是设置绑定账户和设备，以免想刷机再绑定的话就要苦苦等待 7 天了。
  2. 解锁后会清空所有数据，所以要做好备份，我备份的：微信，QQ 聊天记录 (PC 端有备份聊天记录到电脑的功能，后期在从 PC 还原到手机就行)，联系人 (可在软件里面导出为 vcf 文件并拷到电脑上)，照片 (我用的 Google 相册，自动云备份)。

开始刷机

前言

  1. 建议将所有刷机所需要的文件放到同一个文件夹下
  2. 打开 cmd 时确保已经将当前文件夹设置为工作目录。
    方法：(下方任选一个)
    (1) 在资源管理器上方地址栏输入 cmd 回车打开即可
    (2) 按住 Shift，在资源管理器空白处右键单击选择 在此处打开 Powershell 窗口，输入 cmd 回车。(Windows 7 系统可能显示的是 在此处打开命令提示符，直接点击就行)
    (3) Win+R 打开运行，输入 cmd 回车，在 cmd 窗口输入 cd /d "文件夹路径" 回车。

安装 USB 驱动和 ADB

  1. 下载 Google USB 驱动，文件名为 usb_driver_r13-windows.zip，解压后会得到以下文件/夹：amd64、i386、android_winusb.inf、androidwinusb86.cat、androidwinusba64.cat、source.properties
  2. 右键点击 android_winusb.inf，选择 安装，在弹出的对话框点击 安装 即可。
  3. 下载 ADB，文件名为 platform-tools_rXX.X.X-windows.zip (X 为版本号)，解压到任意文件夹后会得到下列文件：adb.exe、AdbWinApi.dll、AdbWinUsbApi.dll、dmtracedump.exe、etc1tool.exe、fastboot.exe、hprof-conv.exe、libwinpthread-1.dll、make_f2fs.exe、make_f2fs_casefold.exe、mke2fs.conf、mke2fs.exe、NOTICE.txt、source.properties、sqlite3.exe。
  4. 运行 ADB 的话，你可以在当前位置打开 cmd (命令提示符)，输入 adb，查看是否可以运行；也可以将这 15 个文件复制到 C:\Windows\System32\ 文件夹里或在系统变量的 PATH 里，将 ADB 所在文件夹添加进去。(三个方法任选一个)

解锁 BootLoader (BL)

  1. 开启 OEM 解锁：开发者选项 >OEM 解锁
  2. 开启 USB 调试：开发者选项 >USB 调试
  3. 开启 USB 调试 (安全设置)：开发者选项 >USB 调试 (安全设置)
  4. 用数据线连接上电脑，打开 cmd 执行 adb devices，在手机上弹出的 是否允许 USB 调试 对话框点击 是。
  5. 执行以下命令让手机进入 FASTBOOT
  6. 打开 小米手机解锁工具，登录账户，按提示解锁手机。
  7. 解锁完成后，手机会自动重启，这时会进入一个全新的 MIUI 系统。
  8. 进入 开发者选项 (需要先手动开启)，重新打开 USB 调试，USB 调试 (安全设置)

  2. 使用任意解压缩软件打开 PixelExperience 的安装包，解压出里面的 payload.bin 待用
  3. 输入以下命令解包 payload.bin，因为我们只需要 boot.img，所以不用等待全部解包完毕，当看见 boot 后面的进度条跑完后，就可以使用 Ctrl+C 关闭程序了，然后在 payload.bin 所在的文件夹里会出现一个叫 extract_YYYYMMDD_HHMMSS 的文件夹，打开它就可以看见 boot.img 了

修改 boot.img

  1. 通过任意方式将 boot.img 传输到手机上你能找到的文件夹，比如 Download 文件夹等。推荐使用数据线传，我用的是：手机端开启 MT 管理器的远程管理功能，电脑端使用 FileZilla 通过 FTP 连接手机来传输文件
  2. 手机下载安装 Magisk
  3. 打开 Magisk，点击右上方的 安装，选择 选择并修补一个文件，选择拷过来的 boot.img，然后 Magisk 就会自动处理并输出到系统的 下载 文件夹(/storage/emulated/0/Download/)，文件名为 magisk_patched-XXXXX_XXXXX.img

Magisk 设置与模块

设置

  1. 打开 Magisk，点击右上方的 齿轮 图标进入设置，找到并打开：Zygisk、生物识别验证。
  2. ROOT 完成的系统可能无法打开个别银行 APP、国家反诈中心等，可以在 Magisk 设置里面找到 排除列表，点击有 ROOT 检测的 APP 的名字，在展开的下拉列表把对应的所有的进程打开。

不要直接点击 APP 后面的选项框，那并不会对这个 APP 的所有进程适用，所以要展开并一个个地打开。

模块

  1. 模块安装很简单，在手机上就可以完成，不用电脑。从提供的链接下载模块，后缀名一般为 .zip，然后在 Magisk 主界面点击右下角的 模块，点击上方的 从本地安装，选择下载好的 zip 文件即可。若要同时安装多个模块可以安装完一个后不点重启，点击左上角的返回，继续安装，最后再重启
  2. 模块推荐：
    (1) 神仙自动救砖
    顾名思义，这是一个防止刷错模块导致无法启动来进行自动救砖的模块。
    原文链接：https://www.coolapk.com/feed/33911784?shareKey=ZGY1YTc2Mjk4NTM1NjI0ZTc0YzQ~
    下载地址：https://wwb.lanzoub.com/ifEj401spt8h
    !!! 下载地址可能不会收到该模块的后期更新，最好到原文链接的评论区查找最新下载链接。这位博主也挺逗的，放了个墓碑机制模块的链接而不是救砖模块的链接，注意不要下载错了，不过也可以在回复里面找到正确的下载地址 !!!
    (2) Universal GMS Doze
    可以防止在没有科学上网的前提下来关闭 Google 服务防止其在后台不停的向 Google 服务器发送请求导致卡顿和费电。当要使用 Google 软件时，会自动运行 Google 服务，以确保不会影响正常使用。
    下载地址：https://github.com/gloeyisk/universal-gms-doze/releases
    (3) Uperf
    合理调度硬件资源延长续航。
    下载地址：https://github.com/yc9559/uperf/releases
  3. 以上模块安装完重启即可生效，无需配置。
  4. 每安装完一个模块都要重启才会生效。

LSPosed 介绍与模块

介绍与安装

  1. LSPosed 倾向于软件修改，Magisk 倾向于系统内部修改。
  2. LSPosed 是作为 Magisk 的模块进行安装的，从文章开头的链接里下载 LSPosed-vX.X.X-XXXX-zygisk-release.zip (不是 riru 版，因为我并没有推荐过这个模块)，打开 Magisk，在主界面右下角点击 模块，选择 从本地安装，选择下载的 zip 文件，等待安装完成，然后重启手机。
  3. 重启之后进入桌面，系统会提示是否将一个指向 LSPosed 的图标添加到主屏幕，添加即可。若没有提示或添加失败，检查 LSPosed 是否已正常安装，可尝试重新安装，若已经正常安装，打开 电话，在拨号盘输入 *#*#5776733#*#* 即可打开。

模块

  1. 与 Magisk 不同， LSPosed 的模块格式为 .apk，可以直接当做普通软件进行直接安装，安装完成会在应用列表显示图标，打开就是对该模块的设置，同时也要到 LSPosed 主界面下方的 模块 里面启用该模块 (一般默认启用)，点击安装的模块还可以设置对那些软件生效 (建议保持默认)。
  2. 模块推荐：
    (1) 短信验证码自动填充
    下载地址：https://github.com/tianma8023/XposedSmsCode/releases
    (2) QAuxiliary
    招牌功能有 QQ 消息防撤回、界面优化等。安装完成激活后重启，打开 QQ >设置 >QAuxiliary 即可进入设置页面
    下载地址：https://github.com/cinit/QAuxiliary/releases
    (3) 不要竖屏
    强制将哔哩哔哩的竖屏视频横屏播放
    下载地址：https://github.com/WankkoRee/Portrait2Landscape/releases
    (4) 拒绝强制亮度
    当点击付款码、收款码、扫描二维码的时候，屏幕会瞬间拉到最高，非常的废眼。下载安装激活后勾选需要生效的应用程序即可
    下载地址：https://github.com/Xposed-Modules-Repo/com.fankes.refusebrightness/releases
