新电脑安装 Windows 10 详细教程

  • 发布时间：2021 年 07 月 11 日
  1. 首页
  2. 正文  

一、进入 BIOS 必须更改的设置

1. 调整界面语言为简体中文 (Chinese Simpfiled)
2. 开启 CPU 加速频率 (Turbo Mode)
3. 高频内存 (2666 MHz以上) 要开启 XMP 模式
4. 开启 CPU虚拟化技术，Intel CPU为 Intel(R) Virtualization Technology (VT-x)和 VT-d，AMD CPU为 SVM Mode
5. 机器若安装了SATA硬盘，需更改SATA工作模式(SATA Mode)为AHCI，同时把带有系统的硬盘设置为第一启动项

更改完成后按键盘上的 F10，再按 Enter 保存设置并重启

更改原因：
1. 便于查看 BIOS 各设置
2. 开启睿频可以提高 CPU 运算速度
3. 开启 XMP 模式以使用更高的内存频率提高运行速度
4. 运行安卓模拟器、VMware 虚拟机等软件要求 CPU 支持虚拟化技术
5. AHCI 模式可以大大提升 SATA 硬盘的读写性能，可能对 M.2 NVMe 硬盘无效

操作摘要：
傲梅分区助手：
1.在要安装系统的硬盘上创建一个分区，文件系统 NTFS，卷标随意，并勾选与 SSD 4K 分区对齐，提交更改
2.点击硬盘，在右侧选择 重建MBR，默认的就行。
3.点击分区，在右侧选择 设置为活动分区，提交更改。

WinNT Setup：
1. 安装源选择下载的ISO文件，会自动挂载改镜像 (因为拍视频所使用的是虚拟机所以要手动选择 ISO文件\sources\install.wim)
2.引导和安装驱动器都设置为要安装系统的分区
3.要安装的系统选择 Windows 10 专业版
4.点击安装，完成后重启。
